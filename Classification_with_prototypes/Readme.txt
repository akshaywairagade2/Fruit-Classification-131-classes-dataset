Training 

It was done on full dataset taking the average rgb values of the images.Ignored the white places to remove the background 
for more accuracy.
Trained data was stored in a list.

Test
Distance was find between the category wise rgb value and image rgb value.If the minimum distance rgb value matches 
the image category image is said to be matched.

Accuracy
Overall accuracy was obtained as 51.097496473906915
Category wise accuracy is shown in output file.
